# DiscordJS Bot

Bot in DiscordJS, used to fetch posts from reddit. Currently has the following commands

```
/hot <limit>
/top <time> <limit>
/random <limit>
\`\`\`

Example: \`/random 5\` \`/top year 20\`
```

# Setup

Copy `.env.example`, rename it as `.env`, and fill in the required information. Then,

```
npm install
npm run start
```
